ITAM (IT Asset Management) adalah aplikasi web manajemen aset IT yang dirancang untuk membantu perusahaan mencatat, melacak, meminjamkan, dan memelihara aset IT secara efisien. Sistem ini menggunakan dark theme premium dengan UI modern dan responsif.


Kemampuan utama mencakup:

  • Pencatatan aset IT lengkap (laptop, printer, monitor, dll) dengan QR Code otomatis.
  • Manajemen toko/lokasi dan kategori aset secara terstruktur.
  • Manajemen karyawan dengan riwayat peminjaman aset.
  • Sistem check-out/check-in (peminjaman & pengembalian) aset ke karyawan.
  • Riwayat maintenance/perbaikan aset dengan tracking biaya dan status.
  • Audit trail/log aktivitas otomatis untuk setiap perubahan data aset.
  • Dashboard analitik visual dengan grafik & statistik real-time.
  • Import & export data aset melalui file Excel (.xlsx).
⚡

Fitur-Fitur Utama

Seluruh kapabilitas yang tersedia dalam sistem

📦

Manajemen Aset IT

CRUD aset lengkap: nama, merek, model, serial number, spesifikasi teknis, kondisi (Good/Fair/Poor/Damaged), status (Active/Inactive/Maintenance/Disposed), harga pembelian, tanggal garansi, foto, catatan, dan lokasi detail.

📱

QR Code Otomatis

Setiap aset yang didaftarkan secara otomatis mendapatkan QR Code unik. QR Code dapat diunduh dan dicetak untuk pelabelan fisik aset. Scanning QR menampilkan detail aset langsung.

🔄

Check-out & Check-in Aset

Sistem peminjaman aset ke karyawan (Check-out) dan pengembalian ke gudang (Check-in) dengan pencatatan catatan serah terima, tanggal, dan admin yang memproses.

🔧

Riwayat Maintenance

Pencatatan riwayat perbaikan/pemeliharaan aset: kendala, teknisi/vendor, tanggal mulai/selesai, biaya, status (Scheduled/In Progress/Completed/Cancelled), dan solusi perbaikan.

👥

Manajemen Karyawan

Pencatatan data karyawan (nama, email, departemen, telepon) dengan auto-generated employee code (EMP-XXXX). Halaman edit menampilkan daftar aset dipegang saat ini dan riwayat penugasan aset lengkap.

📝

Audit Trail / Log Aktivitas

Setiap perubahan data aset (create/update/delete/assign/return) tercatat otomatis. Halaman Log Aktivitas global memungkinkan pencarian, filter berdasarkan aksi dan tanggal, dan export ke Excel.

📊

Dashboard Analitik

Ringkasan visual real-time: total aset, total toko, total kategori, aset rusak, distribusi kondisi, distribusi status, top 5 kategori, top 5 toko, 10 aset terbaru, dan 5 aset berkondisi buruk.

📥

Import & Export Excel

Export seluruh data aset ke file Excel (.xlsx). Download template Excel untuk import massal. Import data aset dari file Excel dengan validasi otomatis (category_code & store_code matching).

🏪

Manajemen Toko/Lokasi

CRUD toko dengan kode unik, nama, lokasi, dan region. Delete protection: toko yang masih memiliki aset tidak bisa dihapus.

🏷️

Manajemen Kategori

CRUD kategori aset dengan kode unik (NTB, MON, PRT, dll.) dan deskripsi. Delete protection: kategori yang masih digunakan oleh aset tidak bisa dihapus.

🔍

Sorting & Filtering

Multi-column sorting pada tabel aset, karyawan, dan toko. Filter pencarian global. Sorting berdasarkan umur aset (kolom added_at), aset dipegang, departemen, dll.

📄

Paginasi Dinamis

Opsi per halaman (10/25/50) pada tabel Daftar Aset, Daftar Karyawan, dan halaman Log Aktivitas. Query string terpreservasi saat mengubah jumlah per halaman.

Keamanan & Akses

Mekanisme keamanan yang diterapkan

🔐 Autentikasi
  • Login berbasis username & password
  • Password di-hash menggunakan bcrypt
  • Middleware auth melindungi semua route kecuali login
  • Redirect otomatis ke /login jika belum terautentikasi
🛡️ Proteksi Data
  • CSRF protection pada semua form POST/PUT/DELETE
  • Validasi input server-side pada setiap controller
  • Delete protection: toko & kategori yang memiliki aset tidak bisa dihapus
  • Foreign key constraint pada level database
📝 Audit Trail
  • Setiap perubahan aset tercatat via AssetObserver
  • Mencatat: siapa, kapan, apa yang berubah (old/new values)
  • Format JSON pada kolom properties
  • Tidak bisa dihapus/dimanipulasi oleh user biasa
